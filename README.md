# Realtor AI MVP

Production-ready real estate agent AI platform.

## Quick Start

```bash
pip install -r requirements.txt
python realtor_mvp.py
```

Visit http://localhost:8080

## Features

- MLS Listing Description Generator
- CMA Report Generator  
- Follow-up Email Generator
- Zero API costs (CLI wrapper proxy)
